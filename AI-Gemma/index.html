人工智能：Gemini/Gemma模型部署与使用

Posted on 2024-07-11 Edited on 2024-07-11 In Artificial Intelligence Waline:

Gemini是Google开发的一款多模态大模型，能够处理文本、图像、音频、视频和代码等信息。目前推出的Gemini模型分为Nano、Pro、Ultra以及1.5Pro，这些模型都可以在谷歌网站上进行访问：https://gemini.google.com 。此外，谷歌还提供了Gemini模型的API，可在代码中调用模型，输入文本和图片然后，输出文本回复，以及一款开源大语言模型gemma，该模型基于Gemini的研究和技术开发，能处理文本信息。模型有2b和7b两种参数规模以及经过调整（7b-it）和未调整的基础模型等版本，可通过多种框架构建：Keras、Pytorch、Transformers、Gemma C++、TensorRT-LLM、TensorFlow Lite、MaxText、Pax、Flax。

部署方法

文档中介绍了在Linux下使用docker配置环境并运行模型的方法，但未说明对模型进行调整的操作和对训练集的要求，推测使用常规Pytorch对NLP模型的训练方式即可。

Keras框架

下载地址同上，可选四个版本的模型。

在Keras框架的配置较为简单，仅需keras和keras-nlp两个包、几行代码即可实现模型的导入、使用、训练和保存，使用代码示例见网站Gemma|Kaggle。

千层Transformers模型部署使用手册

1000 Transformers是Hongyu Wang等人开发的一种将Transformer网络扩展至1000层的技术，论文见 [2203.00555] DeepNet: Scaling Transformers to 1,000 Layers (arxiv.org) ，作者未给出1000层Transformer网络的实现代码和训练过的模型，但提供了一个工具包Torchscale（microsoft/torchscale: Foundation Architecture for (M)LLMs (github.com)）的链接，该工具包结合了包括DeepNet在内的多种技术，可根据需要修改配置代码，构建自己需要的网络结构。

部署方法

根据Torchscale在Github上的安装指南安装该包，仅需几行代码即可构建模型。

例：
git clone https://github.com/microsoft/torchscale.git

cd torchscale
pip install -e .

from torchscale.architecture.config import EncoderConfig
from torchscale.architecture.encoder import Encoder

config = EncoderConfig(vocab_size=64000)
model = Encoder(config)

如上，可通过EncoderConfig类的参数修改网络的层数等。
